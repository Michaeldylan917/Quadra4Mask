# QuadTask 快速开始指南

## [object Object]分钟快速上手

### 第一步：打开应用
1. 在浏览器中打开 `index.html`
2. 您将看到所有原型页面的导航菜单
3. 点击任意页面链接即可查看该页面

### 第二步：浏览启动流程
1. 点击"启动页"查看应用启动界面
2. 点击"隐私声明"了解数据保护承诺
3. 点击"新手引导"学习基本概念

### 第三步：体验主功能
1. 点击"主视图"进入四象限任务管理界面
2. 查看四个象限的任务分布
3. 点击任务卡片查看详情
4. 点击"+"按钮快速添加新任务

### 第四步：探索其他功能
- **日历视图**: 按日期查看任务
- **时间线视图**: 按时间顺序查看任务
- **专注模式**: 使用番茄钟提高效率
- **统计分析**: 查看任务完成情况
- **设置**: 管理应用设置和数据

## 🔧 开发者指南

### 项目结构
```
prototype/
├── index.html          # 主入口
├── css/               # 样式文件
│   ├── global.css     # 全局样式
│   └── components.css # 组件样式
├── js/                # 脚本文件
│   └── app.js         # 应用逻辑
└── pages/             # 页面文件
    ├── splash.html
    ├── privacy.html
    ├── onboarding.html
    ├── home.html
    ├── add-task.html
    ├── task-detail.html
    ├── calendar.html
    ├── timeline.html
    ├── focus.html
    ├── stats.html
    ├── settings.html
    └── help.html
```

### 修改样式
1. 编辑 `css/global.css` 修改全局样式
2. 编辑 `css/components.css` 修改组件样式
3. 使用CSS变量快速调整颜色和间距

### 修改功能
1. 编辑 `js/app.js` 修改应用逻辑
2. 在各个页面的 `<script>` 标签中添加交互代码
3. 使用 LocalStorage API 管理数据

### 添加新页面
1. 在 `pages/` 目录创建新的HTML文件
2. 复制现有页面的基础结构
3. 在 `index.html` 中添加导航链接

## 🎨 自定义主题

### 修改颜色
编辑 `css/global.css` 中的 CSS 变量:

```css
:root {
    --color-primary: #007AFF;              /* 主色 */
    --color-urgent-important: #FF3B30;     /* 重要&紧急 */
    --color-not-urgent-important: #FF9500; /* 重要&不紧急 */
    --color-urgent-not-important: #34C759; /* 不重要&紧急 */
    --color-not-urgent-not-important: #A2845E; /* 不重要&不紧急 */
}
```

### 修改字体
```css
:root {
    --font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif;
    --font-size-base: 16px;
}
```

```css
    --spacing-lg: 24px;
}
```

## 🐛 常见问题

### Q: 数据丢失了怎么办？
A: 检查浏览器是否禁用了 LocalStorage。如果有备份文件，可以通过导入功能恢复。

### Q: 为什么某些功能不工作？
A: 确保浏览器支持 LocalStorage 和 JavaScript。尝试清除浏览器缓存后重新加载。

### Q: 如何在移动设备上使用？
A: 在移动设备的浏览器中打开 `index.html`，或将其部署到服务器后通过URL访问。

### Q: 能否在多个设备间同步数据？
A: 当前版本不支持云同步。可以通过导出/导入功能在设备间转移数据。
